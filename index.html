Version 2.5.2 is now available. Major new features include:

  • Improved screening calculations
  • Better core-level shift calculations
Enjoy!

OCEAN Summer School: 12-14 July 2018

Free, 3-day workshop focused on computational X-ray spectroscopy and the OCEAN code to be held at SLAC National Accelerator Laboratory in California. Click here for details and the schedule. The workshop is over, but slides and examples from the tutorials are available for download from the site under the 'Tutorials' tab.

27 June 2017

Version 2.1 is now available. Major new features include:

  • Faster QE support
  • Cleaned up inputs and defaults
Enjoy!

  — the OCEAN team

23 February 2017

Version 2.0 is now available. Major new features include:

  • RIXS (and valence/UV) now integrated into a single BSE executable alongside the core.
  • General bug fixes, improved FFT options, QE6 support
Enjoy!

  — the OCEAN team

25 October 2016

Version 1.2 is now available. Major new features include:

  • Fixed up LDA+U support (QuantumEspresso only). Thanks to Yufeng Liang for spearheading this effort
  • Improved QE support
  • General bug fixes, including better MPICH support
Enjoy!

  — the OCEAN team

20 October 2015

We are happy to announce the release of version 1.1. Major new features include:

  • Parallel BSE solver!
  • Core-level shifts
  • Improvements to the input and defaults
  • General bug fixes
Enjoy!

  — the OCEAN team
